# Executive Weekly KPI Report

## Overview
Built an automated weekly KPI report using transactional retail data (541k+ rows) to produce executive-ready insights.

## KPIs
- Total Revenue
- Total Orders
- Average Order Value (AOV)
- Weekly trends

## Tools Used
- Python
- Pandas
- Matplotlib
- Excel (export)

## Deliverables
- Jupyter notebook (data cleaning, analysis, visualization)
- Excel report with KPI summary and weekly breakdown

## Key Takeaways
- Revenue and order volume fluctuate week-to-week
- AOV shows variability independent of order count
- Weekly aggregation enables executive-level trend analysis
- Weekly aggregates were validated against total revenue and order counts to ensure accuracy
